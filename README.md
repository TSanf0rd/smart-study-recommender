## Project Description
The Smart Study Resource Recommender is an AI-powered platform designed to help university students efficiently discover personalized study materials. The system uses collaborative filtering and semantic search with embeddings to recommend relevant resources including videos, notes, PDFs, and online materials based on students' courses, learning styles, and uploaded content.

## Project Objectives
- Design a recommendation system that adapts to each student's courses and preferences
- Develop a hybrid model using collaborative filtering and semantic search with embeddings
- Provide students with a platform to upload notes or ask questions and receive targeted study materials
- Evaluate system effectiveness through user feedback and testing

## Technology Stack
- **Backend:** Python, FastAPI
- **Frontend:** JavaScript, React
- **Database:** PostgreSQL with pgvector extension
- **ML/NLP:** Sentence Transformers, scikit-learn
- **Tools:** Docker Compose, GitHub, ClickUp

## Key Features
- User profile management for students, instructors, and tutors
- Note upload and semantic search functionality
- Personalized resource recommendations
- Study progress tracking
- Curated resource management for instructors
- Collaborative study playlists
